# SistemaStock

Sistema de stock en C# (consola) con manejo de productos y ventas.

## Descripción

Este proyecto es un sistema de stock básico donde puedes:

- Agregar productos (nombre, stock y precio)
- Ver lista de productos
- Vender productos (se resta automáticamente del stock)
- Validaciones de entradas para evitar errores

Ideal como proyecto de práctica para C# y para mostrar en GitHub/portafolio.

## Cómo correrlo

1. Tener instalado [.NET SDK](https://dotnet.microsoft.com/en-us/download)  
2. Abrir la terminal en la carpeta del proyecto  
3. Ejecutar:

```bash
dotnet run
